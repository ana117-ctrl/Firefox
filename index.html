Características Principales


Sus características más destacadas son:
Navegación con pestañas. Se pueden abrir simultáneamente varias páginas web de tal forma que cada una se visualiza en una pestaña independiente.
Restauración de sesión. Cuando Firefox se cierra o reinicia se ofrece la opción de restaurar la sesión para evitar la pérdida de información en formularios, descargas.
Corrector ortográfico. Si se dispone del complemento Diccionario de Español/España se puede activar el corrector ortográfico.
Sugerencias de búsqueda. Al comenzar a escribir en la barra de búsqueda de Google se mostrará una lista de sugerencias Accesibilidad: Es un navegador completamente gratuito y está disponible para Windows, iPhone, iPad y Mac


Transparencia


Firefox ofrece varias características y políticas que promueven la transparencia tanto en términos de privacidad como de código abierto. Aquí algunos puntos destacados:

  1. Código Abierto: Firefox es desarrollado por Mozilla, una organización sin fines de lucro, y su código fuente es completamente abierto. Esto significa que cualquiera puede:
    • Revisar el código para verificar cómo funciona.
    • Contribuir al desarrollo del navegador.
    • Usar y modificar el código según las licencias abiertas.
  2. Privacidad y Datos del Usuario: Recopila solo la información necesaria para mejorar la experiencia del usuario y permite ver qué datos se recopilan y ajustar configuraciones para limitar el rastreo.
  3. Información Pública: Publica informes y documentación sobre el estado del navegador, iniciativas de privacidad y seguridad, y actualizaciones y cambios realizados en el software.
  4. Herramientas de Control para el Usuario:Firefox pone a disposición de los usuarios herramientas para gestionar y entender mejor cómo se utiliza su navegador
  5. Comunidad Activa: La comunidad global de desarrolladores, testers y usuarios participa activamente en la mejora del navegador.



Seguridad


Algunas de las funciones de seguridad de Firefox son:


  1. Bloqueo de rastreadores de terceros, redes sociales, criptomineros y lectores de huellas digitales
  2. Protección contra el rastreo mejorada
  3. Bloqueo de sitios de ataque registrados
  4. Bloqueo de falsificaciones web registradas
  5. Notificaciones de intentos de instalación de complementos

Costos


El navegador Firefox es gratuito y no tiene costos ocultos. Mozilla, la fundación que respalda a Firefox, no vende los datos personales de los usuarios.


Propósito


Su objetivo es que internet sea un recurso que esté al alcance de todos, y que las personas puedan ser contribuyentes y creadores de la web


Tipo de licencia


Mozilla Firefox está bajo la licencia pública de Mozilla (MPL). Esta licencia es de tipo Open Source (código abierto). La MPL es una licencia de software libre que otorga diferentes privilegios a los usuarios y desarrolladores. La Fundación Mozilla es la encargada de desarrollarla y mantenerla.


Requisitos para contribuir


Tener conocimiento en lenguajes de Programación, principalmente C++, JavaScript, Rust y Python. Ademas qué saber utilizar HTML, CSS y estándares web puede ser muy útil al momento de editar el navegador. Seguir las guías para la edición, Mozilla tiene una guía detallada de desarrollo: "Firefox Source Docs". Lee y sigue las pautas de estilo de código.


Cómo puedes contribuir…
Puede traducir contenido, crear interfaces más accesibles e iconos para el navegador, o incluso puedes ayudar proando versiones nuevas de Firefox y reportar los errores.
